# Advent of Code 2025

This repository contains my solutions to the [Advent of Code 2025](https://adventofcode.com/2025) programming challenges. Advent of Code is an annual event featuring daily algorithmic puzzles that test problem-solving skills, algorithmic thinking, and code optimization.

## Structure

- `solutions/` - Python implementations for each day's challenge
- `inputs/` - Problem input files and test cases

## Technologies

- **Python** - Primary language for initial solutions
